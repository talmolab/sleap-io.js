# Demo

## Local development

1. Build the package:

```bash
npm run build
```

2. Serve the repo root (needs static server for module imports):

```bash
npx serve -p 8080 --cors --no-clipboard
```

3. Open the demo page:

```
http://localhost:8080/demo/index.html
```

The demo is preloaded with `demo/assets/demo-flies13-preds.slp` and `demo/assets/demo-flies13-preds.mp4`.
